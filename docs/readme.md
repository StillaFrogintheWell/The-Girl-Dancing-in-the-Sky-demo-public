# \[Dreamfall Sky]

A short 3D dream-flight game made with Godot.

You drift through a dreamy sky, pass through clouds, avoid obstacles, and chase a soft, emotional atmosphere rather than intense action. This project was created as a small prototype, but it already captures the feeling of flying through a dream.

## Play the Demo

[Play on Demo Pages](https://stillafroginthewell.github.io/The-Girl-Dancing-in-the-Sky-demo-public/)


## About the Game

This is a simple atmospheric flying game set in a dreamlike sky world.

The goal is to keep flying, move through the air smoothly, and experience the mood of the world. Along the way, you may encounter clouds, floating islands, and other objects that affect the feeling of the run.

The game focuses on:

* dreamlike atmosphere
* simple movement
* light 3D gameplay
* emotional tone over complexity

## Features

* 3D sky-flight gameplay
* Soft dream-inspired presentation
* Clouds and floating objects
* Lightweight browser-playable build
* Ending images for clear / game over states
* Made with Godot
